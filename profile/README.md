## Why This Lab Exists

Look at [what's out there](https://github.com/Wall-of-Shames) (till bilding). One word: SHAME! 

API wrappers sold as eg. "advanced AI hacking tools". eg. WormGPT clones with Telegram payment funnels. Repos with thousands of forks built on stolen code and fake stars. Non-technical users ‚Äî often young, often beginners ‚Äî getting scammed by people who discovered that slapping "GPT" on a `requests.get()` call makes easy money. And if i bust them! Ej is opensource! WTF! What? Let us wanna pul thhe crap!

This is what happens when AI becomes easy to fake and hard to understand.

Codey-LAB is the answer to that. Not by shouting louder ‚Äî but by raising the actual floor. If you know what LLMs really are, how they really work, and how to use them as a tool instead of a magic trick, you're immune to the scam. You become the one who spots it.

**We build the knowledge that makes scams obsolete.**

## What Is Codey-LAB?

A curated, invite-based knowledge lab for developers who use AI as a sidekick ‚Äî not a ghostwriter, not a magic oracle, not a product to resell.

We build and maintain:

- **Universal system prompts** that work across Claude, GPT-4o, Gemini, Llama, Mistral ‚Äî without the 6-page outputs nobody asked for
- **AI workflow patterns** for real production environments, built by people who pay API bills
- **Security-first tooling** under the ESOL v1.1 ethical framework
- **GitHub automation** that earns its place in a pipeline

Everything here is battle-tested. Nothing here is tutorial code.

## Lab Areas

### `[AREA: PROMPTS]`
The core of the lab. System prompts engineered to eliminate AI datam√ºll at the source ‚Äî for every major model, tuned for senior dev workflows. No preambles. No summaries. No 500-token answers to a 2-line question.

### `[AREA: SEC]`
Security tooling and prompts under ESOL v1.1. Blue team, authorized red team, code audits, hardening. Allied with [Wall of Shames](https://github.com/Wall-of-Shames) ‚Äî the same team, the other side of the work.

### `[AREA: AI]`
Working integration patterns for real pipelines. Token-efficient architectures, repo context loading, multi-model orchestration. No theory. No tutorials.

### `[AREA: GH]`
GitHub automation that earns its place. Actions patterns, profile tooling, and the Codey ecosystem itself.

### `[AREA: OSS]`
- [Codey](https://github.com/VolkanSah/Codey) ‚Äî GitHub activity pet, gatekeeper, and honest mirror of your work. RPG No Mercy Edition.
- [ESOL](https://github.com/ESOL-License/ESOL) ‚Äî Ethical Security Operations License v1.1. For when Apache 2.0 isn't enough.
